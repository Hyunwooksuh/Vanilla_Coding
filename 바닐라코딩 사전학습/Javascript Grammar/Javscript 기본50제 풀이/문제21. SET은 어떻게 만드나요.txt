
Q. 다음 중 set을 만드는 방법으로 올바른 것을 모두 고르시오.
1)  var x = {1, 2, 3, 5, 6, 7};
2)  var x = {};
3)  var x = new Set('javascript');
4)  var x = new Set(range(5));
5)  var x = new Set();

정답 3,5번 
Set이란 중복이 허용되지 않는 객체로서, 안에 이미 같은 값이 존재한다면
추가되지 않음. 그래서 중복값이 없어야 한다면 Set을 이용하면 됨

Set 생성방법 -> let x = new Set();
값은 배열 값을 포함
